## Glossary

| Term | Definition |
|---|---|
| **Associate Software Engineer** | The role level at Red Hat that corresponds to a recent graduate / entry-level engineer. The competency expectations in this document are calibrated to this level. |
| **CodeFlare SDK** | An open-source Python SDK that simplifies distributed ML workload management on Kubernetes/OpenShift using KubeRay and Kueue. |
| **RHOAIENG** | Red Hat OpenShift AI Engineering — the Jira project tracking engineering work on the RHOAI product suite. |
| **GETDO** | Global Engineering Talent Development & Operations — the Jira project tracking intern/new-hire onboarding and development activities. |

| **MaaS** | Model-as-a-Service — Red Hat's internal platform for serving LLMs such as IBM Granite. |
| **MCP** | Model Context Protocol — an open protocol for connecting AI models to external data sources and tools. |
| **RAG** | Retrieval-Augmented Generation — a technique where relevant documents are retrieved and injected into the LLM prompt at query time to ground responses in factual context. |
| **LanceDB** | An embedded vector database used to store and query document embeddings for RAG workloads. |
| **CVE** | Common Vulnerabilities and Exposures — a publicly disclosed security vulnerability. |
| **Kueue** | A Kubernetes-native job queueing system used to manage workload admission and resource fairness. |
| **PyPi** | Python Package Index — the standard repository for distributing Python packages. |
